# 📚 Programming 1 — Semester 1

Exercises, workshops, exams, and projects from the Programming 1 course, written in **Racket** and developed in **DrRacket 8.15+**.

---

## 🗂️ Repository Structure

### 🧱 [`clase/`](./clase)
Notes and exercises from the first weeks of the semester. Covers the fundamentals: expressions, function organization, and regular expressions. The starting point of the course.

---

### 🔀 [`condicionales/`](./condicionales)
Exercises on conditional structures using `if` and `cond`. Includes workshops with conditional algorithms, grade tables, and interactive menu exercises.

---

### ⚙️ [`funciones/`](./funciones)
Basic exercises on defining and using functions in Racket. Covers arithmetic operations, homework assignments, and first practices of functional programming.

---

### 🔤 [`strings/`](./strings)
String manipulation exercises. Includes operations over text, character handling, and exercises involving letter sizing.

---

### 🏗️ [`struct/`](./struct)
Exercises with user-defined data structures using `struct`. Defines, creates, and inspects compound records.

---

### 🔁 [`recursividad/`](./recursividad)
Recursion exercises ranging from basic recursive functions to more advanced problems like prime numbers and ASCII pyramids. Includes recursion workshops.

---

### 📊 [`vectores/`](./vectores)
Exercises using vectors and arrays in Racket. Covers vector operations, tabular data, and a histogram implemented with vectors.

---

### 📝 [`parcial/`](./parcial)
Midterm exam files from the semester. Topics covered include conditionals, functions, lists, and vectors.

---

### 🛠️ [`taller/`](./taller)
General practical workshops: algorithms with `if` and functions, asterisk figures, and exercises that integrate multiple topics.

---

### 🎨 [`graficas/`](./graficas)
Graphical programs using the `graphics` and `2htdp/image` libraries. Draws geometric shapes, polygons (decagon), and windows with figures.

---

### 🎮 [`juegos/`](./juegos)
Interactive game projects built in Racket. Includes a ball-dodging game with collision detection, player movement, and win/lose conditions.

---

## 🛠️ Requirements

- [DrRacket 8.15+](https://racket-lang.org/)
- Open any `.rkt` file in DrRacket and press **Run** (or `F5`)

> Files in WXME format (containing embedded images or comment boxes) can only be edited inside DrRacket. `.bak` files are automatic backups generated by the IDE — safe to ignore.

---

## 📌 Language & Libraries

All code is written in **Racket** using a functional programming paradigm. Main libraries used:

| Library | Purpose |
|---|---|
| `(lib "graphics.ss" "graphics")` | 2D graphics viewport |
| `2htdp/image` | Image and shape creation |
| `2htdp/universe` | Animations and interactivity |
